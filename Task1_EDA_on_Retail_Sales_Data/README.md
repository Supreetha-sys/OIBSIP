# Exploratory Data Analysis (EDA) on Retail Sales Data

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a Retail Sales dataset using Python. The objective is to analyze customer purchasing behavior, sales performance, and product trends through data visualization and statistical analysis.


## Objective
- Analyze retail sales data.
- Understand customer demographics.
- Identify sales trends and patterns.
- Generate meaningful business insights using data visualization.


## Tools and Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Dataset
**Dataset:** Retail Sales Dataset (.csv)

The dataset contains information such as:
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount


## Analysis Performed
- Imported required Python libraries
- Loaded the dataset
- Explored dataset structure
- Checked dataset shape and information
- Identified missing values
- Generated descriptive statistics
- Analyzed customer gender distribution
- Analyzed product category distribution
- Visualized age distribution
- Calculated total sales by product category
- Analyzed monthly sales trends
- Examined the relationship between age and total spending
- Performed correlation analysis
- Created a correlation heatmap


## Key Insights
- Clothing has the highest number of customer transactions.
- Electronics generated the highest total sales revenue.
- May recorded the highest monthly sales.
- September recorded the lowest monthly sales.
- Customer age has a very weak relationship with total spending.
- Price per Unit has a strong positive correlation with Total Amount.
